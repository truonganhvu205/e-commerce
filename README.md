# FakeStoreAPI
https://github.com/keikaavousi/fake-store-api

# Clone project
```bash
git init
git clone https://github.com/truonganhvu205/e-commerce.git
cd e-commerce
npm install
```

# Libraries
## React Icons
```bash
npm install react-icons --save
```

## use-local-storage-state
```bash
npm install use-local-storage-state
```

# Getting Started
```bash
npm run dev
```
